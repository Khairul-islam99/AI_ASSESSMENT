# AI Assessment

USING FastAPI · LangChain · LangGraph · MCP · Qwen2.5 3B (local) · PostgreSQL

##  Installation
### 1. Clone the Repository
```bash
git clone https://github.com/Khairul-islam99/AI_ASSESSMENT.git
```
### 2. Create a Virtual Environment (Conda)
conda create -n ai_env python=3.12 -y

conda activate ai_env

### 3. Install Dependencies
pip install -r requirements.txt

### 4. Run the Backend (FastAPI Server)
uvicorn main:app --reload
